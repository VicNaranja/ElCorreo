# El Correo News

Digest diario de noticias de [elcorreo.com](https://www.elcorreo.com) organizado por secciones.

## Características

- Noticias agrupadas por secciones (Bizkaia, Política, Deportes, Athletic...)
- Imágenes de cada noticia
- Búsqueda en tiempo real
- Actualización con un clic

## Uso local

Requiere Python 3.9+

```bash
pip install -r requirements.txt
python app.py
```

Se abre automáticamente en `http://localhost:5000`

## Generar ejecutable (.exe)

```bash
build.bat
```

El ejecutable se genera en `dist/ElCorreo.exe`.
